# 🌿 Eternal Garden

**Eternal Garden** is a premium digital memorial sanctuary platform where people can preserve memories, honor loved ones, and create lasting remembrance spaces online.

> **"Death ends life, not memory."**

It combines emotional design, remembrance rituals, family legacy tools, and premium experiences into one modern platform.

---

# ✨ Vision

Most memorial websites feel outdated and cold.

**Eternal Garden** is designed to feel:

- Peaceful  
- Elegant  
- Emotional  
- Sacred  
- Modern  

A place where memory continues to live.

---

# 🕊 Core Features

## Memorial Sanctuary Profiles

Create personalized remembrance pages with:

- Full name
- Birth / death dates
- Biography
- Photo gallery
- Quote
- Family links
- Memories
- Videos

## 🕯 Candle Ritual System

Visitors can light candles in memory.

- Real-time candles
- Anonymous option
- Message attached
- Hover details

## 🎨 Premium Theme System

- Garden Theme
- Marble Theme
- Orthodox Theme
- Eternal Night Theme

## 🐾 Pet Memorial Profiles

- Dogs
- Cats

## 👑 User Roles

- User
- Moderator
- Admin
